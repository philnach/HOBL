    custom_trace

    Periodically start ETL traces.

    Parameters:

    providers - ETL provider files to use Default: power_light.wprp Options: abl_perf.wprp, full_th.wprp, full_th_wpp.wprp, MediaEngine.WPRP, multimedia.wprp, perf_utc.wprp, pmu.wprp, power.wprp, power_Cortana.wprp, power_heavy.wprp, power_heavy_modified - Copy.wprp, power_heavy_modified.wprp, Power_Light.wprp, Power_Light_cpu.wprp, power_memory.wprp, power_with_webview2_updated.wprp, productivity_perf.wprp, stack_walk.wprp, web_perf.wprp

    trace_duration - The duration of the trace in seconds Default: 300

    start_delay - The start delay before the trace starts in seconds Default: 0

    enable_check - Check the result of the trace. 0 = no check, 1 = check Default: 1 Options: 0, 1

    single_run - Run the trace only once. 0 = no, 1 = yes Default: 0 Options: 0, 1
